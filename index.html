📱 QRIS Pembayaran

Masukkan QRIS string dari bank/e-wallet kamu. QR akan muncul otomatis saat kasir pilih metode QRIS.

✅ QR valid — siap dipakai

💡 QRIS string bisa didapat dari aplikasi bank (BCA, BRI, BNI, GoPay, OVO, dll) — pilih Terima QRIS lalu copy string-nya.

⚡ QRIS Real-time ⭐ PRO
🔒
Fitur Premium
QRIS otomatis terkonfirmasi saat pelanggan bayar — tanpa tombol manual
✅ Moota terhubung — QRIS auto-konfirmasi aktif
Panduan Setup Moota
1. Daftar di moota.co → Tambah Rekening (pilih bank kamu)
2. Di Moota: Settings → Webhook → paste URL di atas
3. Isi kolom Secret Token di Moota → isi yang sama di sini
4. Deploy Edge Function (lihat README di repo)
5. Selesai — QRIS langsung auto-konfirmasi saat ada bayaran masuk 🎉
🧾 Pajak & Biaya Layanan

Dikenakan otomatis setelah diskon. Isi 0 untuk menonaktifkan.

💡 Contoh: Service 5% + PPN 11% = tagihan pelanggan lebih tinggi dari harga menu. Sudah masuk di struk otomatis.
